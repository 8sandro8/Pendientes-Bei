# Harmony Clay - Pendientes Artesanales

¡Bienvenido al proyecto **Harmony Clay**! 🌸

Esta es la tienda online de **Pendientes Bei**, un espacio dedicado a la venta de pendientes artesanales únicos. La web está diseñada para ser sencilla, rápida y bonita, permitiendo a los clientes ver el catálogo, filtrar por categorías (Aros, Largos, Elegantes...) y hacer pedidos directamente.

## ¿Qué hace esta web?

### Para los Clientes:
- **Catálogo Visual:** Pueden ver todos los pendientes con fotos grandes y claras.
- **Buscador y Filtros:** Si buscan algo concreto (como "rojo" o "aros"), lo encuentran al momento.
- **Carrito de Compra:** Pueden ir añadiendo lo que les gusta.
- **Pedidos Fáciles:** Al confirmar el pedido, reciben un correo automático con el resumen (y nosotros recibimos otro para prepararlo). ¡Todo al instante!
- **Solicitudes:** Si algo está agotado, pueden pedir que lo hagamos de nuevo con un botón de "Solicitar".

### Para la Administradora (Bei):
- **Panel de Control:** Entrando con contraseña, la web se transforma.
- **Gestión Total:** Se pueden subir fotos nuevas, cambiar precios, editar descripciones o borrar productos antiguos.
- **Gestión de Pedidos:** Se ven todos los pedidos, se puede marcar si están "Enviados" o "Entregados".
- **Cola de Correos:** Implementamos un sistema inteligente para que, si hay muchos pedidos a la vez, la web no se bloquee. Los correos salen ordenadamente en segundo plano.

## Tecnología
Está construida con **Node.js** (backend) y **HTML/CSS/JS** (frontend) puros, sin complicaciones innecesarias, para que cargue volando y sea fácil de mantener.
